# Vim Commands

`w` Goes to the start of the next word \
`b` Goes to the beginning of the current or previous word \
`e` Goes to the end of the word \
`daw` Deletes current word \
`dd` Deletes the whole line \
`u` Undo \
`r` Redo \
`h` Left \
`j` Down \
`k` Up \
`l` Right \
`gg` Start of file \
`G` End of file \
